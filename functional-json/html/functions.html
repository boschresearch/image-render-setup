ISON Functions

ISON Functions#

This is a reference documentation of the available functions in the default ISON parser. Note that additional function modules can be registered with the parser. These functions will be documented in the corresponding module’s documentation.

Note

If you are writing Catharsys configurations, there are some additional functions you can use. See AnyBase ISON Functions.

Contents

  • Conversion Functions
    • bool - Convert to Boolean
    • float - Convert to Float
    • int - Convert to Integer
    • json - Convert to JSON
    • to-ref-path - Convert to Reference Path
  • File/Path Functions
    • dir - List Directory Contents
    • path.* - Path Components
    • read - Read Text/JSON File
    • write - Write Text/JSON File
  • Logic Functions
    • and - Logic AND
    • eq - Equal
    • if - Conditional Parsing
    • not - Logic NOT
    • or - Logic OR
  • Math Functions
    • sub - Subtraction
    • sum - Summation
    • prod - Product
    • mod - Modulo
    • rand.choice - Select Randomly
    • rand.generator - Create a random number generator instance
    • rand.int - Get Uniform Integer Random Number
    • rand.sample - Sample from Set
    • rand.sample_range - Sample from Integer Range
    • rand.seed - Set Random Seed
    • rand.uniform - Get Uniform Random Number
    • rand.zwicky - Create Instance of Zwicky Box
  • String Functions
    • join - Join Strings
    • str - Formatted String
  • Data Structure Functions
    • enumerate - Enumerate Lists
    • group - Group of lists combination
    • range - List of indices
    • sort - Sort a List
    • union - Union of lists or dictionaries
    • len - Length of a List
    • circularselect - List element selection based on modulo circle.
  • Lambda Function Handling
    • L - Define Lambda Function
    • L* - Convert Structure to Lambda Function
    • ! - Explicit Lambda Function Execution
    • !foreach - Execute Lambda Function for Each Element
    • !where - Select with Lambda Condition
  • Special Functions
    • * - Create Structure from String
    • print - Print to Console or Log
    • set-log-path - Set Logging Path
    • S - String
